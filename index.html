Sobre mí

Soy desarrollador con enfoque en análisis de datos, pipelines ETL y automatización de procesos. Me especializo en construir sistemas que toman datos crudos y los convierten en información útil y accionable.

He desarrollado aplicaciones de producción que van desde sistemas de búsqueda vehicular con más de 45,000 registros hasta herramientas de procesamiento inteligente de documentos con IA local. Trabajo cómodo tanto en el backend como conectando capas de datos con interfaces web.

Me interesan especialmente los proyectos que involucren integración de fuentes de datos, automatización con herramientas como n8n, y arquitecturas que escalen en producción con Docker y CI/CD.

Foco profesional

  • Pipelines ETL & procesamiento de datos
  • Backend con Java / Spring Boot / Python
  • Automatización de flujos con n8n y Macros en Excel
  • Integración de bases de datos SQL/NoSQL
  • DevOps con Docker, Azure & CI/CD
  • Dashboards en Power Bi

Sistema dual (web + desktop) para procesamiento y categorización automatizada de documentos mediante NLP y RAG. La versión web funciona 100% offline en el navegador (privacy-first). La app desktop incluye LLM local con Ollama y búsqueda semántica vectorial.
